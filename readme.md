## üî® instructions

to get started with the app, clone the repo:
```
$ git@github.com:firmoholanda/bookstore-rails-api.git
```

install dependencies:
```
$ bundle install
```

start development derver:
```
$ rails s
```

open app on your browser:
```
$ http://localhost:3000/
```

run rubocop:
```
$ rubocop
```

run tests:
```
$ bundle exec rspec
```
